# 赞助商

OpenClaw.NET 欢迎来自个人、公司和组织的赞助，以支持一个实用、可自托管、供应商中立的 .NET Agent 运行时和网关。

## 赞助支持的内容

赞助可以支持：

- 开发
- 文档
- 示例
- 基础设施
- 发布工作
- 社区支持
- 测试和兼容性工作

赞助商帮助维持项目，但赞助不改变项目治理或技术权限。

## 赞助不提供的内容

赞助不提供：

- 路线图控制权
- 发布权
- 维护者身份
- 独占权利
- 生态方向所有权
- 供应商特定默认配置
- 对架构或安全决策的优先权限

维护者身份基于持续贡献、信任、技术判断和社区行为，而非赞助。

## 赞助与商业使用

欢迎下游商业产品。

商业贡献者和赞助商可以在 OpenClaw.NET 之上构建产品、集成、内部平台、适配器、服务或部署产品。上游项目必须保持供应商中立、可自托管和可复用。

当贡献直接支持公司或客户用例时，贡献者应在相关的 Issue、Pull Request 或提案中披露该背景，以便维护者评估范围、中立性、安全态势以及该工作是否属于 core 还是扩展。
